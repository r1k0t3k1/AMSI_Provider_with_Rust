# AMSI_Provider_with_Rust

## build

```bash
$ rustup install nightly
$ rustup default nightly
$ cargo build
```

## Install

In Admin terminal

```powershell
regsvr32.exe AMSI_Provider_with_Rust.dll
```

## Uninstall

In Admin terminal

```powershell
regsvr32.exe /u AMSI_Provider_with_Rust.dll
```
